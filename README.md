# 📚 le-biblioth-que

A sleek, full-stack MERN application designed for comprehensive book inventory management.  
**le-biblioth-que** bridges a robust Node.js backend with a high-performance React frontend to provide a seamless CRUD experience.

---

## 🚀 Features

- Full CRUD operations (Create, Read, Update, Delete)
- Modern UI using Tailwind CSS + Vite
- RESTful API architecture
- Secure environment variable handling
- Efficient state management using React Hooks & Axios

---

## 🛠️ Tech Stack

**Frontend:** React, Tailwind CSS, Vite, Axios  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose  
**Version Control:** Git
